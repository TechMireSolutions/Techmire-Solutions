## 2. Style

**Primary style:** Editorial Minimalism (award-winning agency aesthetic)  
**Secondary:** Dark mode first, light accent sections  
**Reference:** utility.agency — full-bleed sections, large type, single accent color  

### Anti-patterns to avoid
- No box shadows on dark bg (use `border border-white/[0.07]` instead)
- No card hover lifts (`translateY`) — use opacity transitions
- No gradient buttons — flat fill only
- No emoji or icons in body text
- No centered hero text — always left-aligned
- No decorative SVG above `opacity: 0.12`
- No `max-w-[1440px]` — all sections full-width, padded only

## 6. Interaction Patterns

### Arrow buttons (dual-arrow)
Primary arrow exits top-right on hover, secondary enters from bottom-left.
```tsx
<span className="relative overflow-hidden inline-flex w-[13px] h-[13px]">
  <span className="absolute inset-0 ... group-hover:translate-x-2 group-hover:-translate-y-2">↗</span>
  <span className="absolute inset-0 ... -translate-x-2 translate-y-2 group-hover:translate-x-0 group-hover:translate-y-0">↗</span>
</span>
```

### Underline-grow
`w-0 → w-full` absolute line below link text, `duration-[350ms] ease-out`.

### Hover dots (services grid)
`bg-white/15 → bg-orange` on hover, `transition-colors duration-200`.

### Scroll-aware navbar
`scrollY > 40` → `bg-[#fcfdff] + shadow`. Logo: `filter: invert(1)`. Duration: `300ms`.

## 7. Animation Tokens

| Token | Value | Use |
|-------|-------|-----|
| Entrance ease | `[0.16, 1, 0.3, 1]` | All `whileInView` / `animate` entrances |
| Hero entrance | `duration: 0.9, delay: 0.1` | H1 line reveal |
| Section entrance | `duration: 0.8` | `whileInView` blocks |
| Stagger | `delay: i * 0.06` | Grid items |
| Micro-interaction | `duration: 0.3` | Arrow transforms |
| Underline grow | `duration: 350ms` | Nav link underline |
| Color transition | `duration: 200ms` | Hover color changes |

**Reduced motion:** `MotionConfig reducedMotion="user"` wraps the entire app in `LenisProvider` — Framer Motion automatically disables animations for users who prefer reduced motion.

## 9. Accessibility Standards

| Check | Implementation |
|-------|---------------|
| Focus rings | `outline: 2px solid #E8522A; outline-offset: 3px` via `:focus-visible` in globals.css |
| Reduced motion | `MotionConfig reducedMotion="user"` + CSS `prefers-reduced-motion` fallback |
| Touch targets | All interactive elements min `44×44px` via `min-h-[44px]` |
| Arrow icons | `aria-hidden="true"` on decorative arrows |
| Rotating text | `aria-live="polite" aria-atomic="true"` on container |
| Menu dialog | `role="dialog" aria-modal="true"` on overlay |
| Keyboard nav | Escape closes menu, `aria-expanded` on hamburger |
| Viewport meta | `Viewport` export in `app/layout.tsx` |
